Compute the mean difference

\[ \bar{d}  = {\sum{d_i} \over n } = { 3+6 \over 8} \]

Compute the variance of the differences.

\[ s^2_{d}  ={\sum(d_i -\bar{d})^2 \over n-1 } =  { 3+6 \over 9} \]
